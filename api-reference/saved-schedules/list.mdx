---
title: "GET /api/saved-schedules"
description: "Retrieve all saved schedules belonging to the authenticated user."
---

## Overview

Returns every schedule that the authenticated user has saved. Each entry includes the full section details via the `schedulesSections` junction records. Schedules are ordered from most recently created to oldest.

## Request

### HTTP method and path

`GET /api/saved-schedules`

### Authentication

Bearer token required.

```
Authorization: Bearer <accessToken>
```

### Request body

None.

## Response

HTTP `200 OK`

An array of saved schedule objects ordered by `createdAt` descending. Returns an empty array `[]` when you have no saved schedules.

<ResponseField name="[]" type="object[]" required>
  <Expandable title="saved schedule properties">
    <ResponseField name="id" type="string" required>
      UUID that uniquely identifies the saved schedule.
    </ResponseField>
    <ResponseField name="name" type="string" required>
      Display name of the saved schedule.
    </ResponseField>
    <ResponseField name="createdAt" type="string" required>
      ISO 8601 timestamp indicating when the schedule was saved.
    </ResponseField>
    <ResponseField name="userId" type="string" required>
      UUID of the user who owns this saved schedule.
    </ResponseField>
    <ResponseField name="schedulesSections" type="object[]" required>
      Junction records linking this schedule to its sections, each including full section details.
    </ResponseField>
  </Expandable>
</ResponseField>
