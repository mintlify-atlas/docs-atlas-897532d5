---
title: "GET /api/professors"
description: "Retrieve all professors belonging to the authenticated user."
---

## Overview

Returns every professor that was created by the authenticated user. Professors from other users are never included in the response.

## Request

### HTTP method and path

`GET /api/professors`

### Authentication

Bearer token required.

```
Authorization: Bearer <accessToken>
```

### Request body

None.

## Response

HTTP `200 OK`

An array of professor objects. Returns an empty array `[]` when the user has no professors.

<ResponseField name="[]" type="object[]" required>
  <Expandable title="professor properties">
    <ResponseField name="id" type="string" required>
      UUID that uniquely identifies the professor.
    </ResponseField>
    <ResponseField name="fullName" type="string" required>
      Full name of the professor.
    </ResponseField>
    <ResponseField name="userId" type="string" required>
      UUID of the user who owns this professor.
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>

curl --request GET \

## Error responses

| Status | Description |
| ------ | ----------- |
| `401 Unauthorized` | The access token is missing, malformed, or expired. Obtain a fresh token from `/api/auth/refresh`. |
