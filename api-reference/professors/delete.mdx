---
title: "DELETE /api/professors/:id"
description: "Permanently delete a professor."
---

## Overview

Deletes a professor by UUID. The professor must belong to the authenticated user.

<Warning>
  Deleting a professor is irreversible. Any sections that referenced this professor will have their `professorId` set to `null` automatically — the sections themselves are not deleted.
</Warning>

## Request

### HTTP method and path

`DELETE /api/professors/:id`

### Authentication

Bearer token required.

```
Authorization: Bearer <accessToken>
```

### Path parameters

<ParamField path="id" type="string" required placeholder="e5f6a7b8-9c0d-1e2f-3a4b-5c6d7e8f9a0b">
  UUID of the professor to delete.
</ParamField>

### Request body

None.

## Response

HTTP `204 No Content`

The response body is empty on success.

## Example

<CodeGroup>

```bash curl
curl --request DELETE \
  --url https://api.scheduly.app/api/professors/e5f6a7b8-9c0d-1e2f-3a4b-5c6d7e8f9a0b \
  --header 'Authorization: Bearer <accessToken>'
```

```json Response
(empty body — 204 No Content)
```

</CodeGroup>

## Error responses

| Status | Description |
| ------ | ----------- |
| `401 Unauthorized` | The access token is missing, malformed, or expired. Obtain a fresh token from `/api/auth/refresh`. |
| `404 Not Found` | No professor with the given ID exists for the authenticated user. |
